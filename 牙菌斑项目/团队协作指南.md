本项目用 **SERS（表面增强拉曼光谱）+ AI** 诊断牙周炎。

## 第一步：拿到项目后做什么

### 如果你有 Claude Code
```bash
cd <项目根目录>\DentalPlaque_SERS_Toolbox
claude
```
然后告诉 Claude："回到牙菌斑SERS工程，看看当前进度"。
Claude 会自动读取 `CLAUDE.md` 并理解整个项目。

### 如果你用 MATLAB
```matlab
addpath(genpath(pwd))  % run from DentalPlaque_SERS_Toolbox
main              % 一键运行全部流程
main('dashboard') % 只生成单个病人QC报告
main('phase2')    % 只运行Phase 2统计分析
```

## 第二步：共享方式

### 方式A：完整打包（适合新成员首次接手）
将整个 `<项目根目录>\` 打包为一个 zip：
```
<项目根目录>\
├── CLAUDE.md                     ← 项目说明书（Claude自动读取）
├── 牙菌斑背景.docx               ← 论文引言草稿
├── PatientRegistry_医生填写版.xlsx ← 临床登记表
├── 牙菌斑SERS光谱\               ← 原始数据
│   ├── 阳性+\  (31 patients)
│   ├── 阴性-\  (21 patients)
│   └── 其它数据\
│       └── 未知\  (17 patients)
├── DentalPlaque_SERS_Toolbox\    ← MATLAB工具箱
│   └── ...
└── MATLAB\                        ← 早期草稿代码
```

```matlab
cfg.Data.DataRoot = fullfile(projectRoot, '牙菌斑SERS光谱');
```

只需要共享以下文件夹：
1. `DentalPlaque_SERS_Toolbox/` — 代码库

3. `DentalPlaque_SERS_Toolbox/Figures/Phase2/` — 论文级图表

### 方式C：只共享结果（适合医生/合作者查看）
只需要：

## 第三步：多 Claude 协作方案

如果你和团队成员各有 Claude Code 账号，可以这样协作：

## 第四步：常见协作任务

### 医生：填写临床信息
打开 `PatientRegistry_医生填写版.xlsx`，填写黄色单元格：
- 性别、年龄、临床标签(Healthy/Gingivitis/Periodontitis)
- 医生确认改为 Yes
- 红色姓名的患者确认后修改
完成后交回，我们会更新数据库。

### 数据管理员：新增病人数据
1. 把CSV文件放到 `牙菌斑SERS光谱\<组名>\<患者ID>\`
2. 运行 `main('read')` 重建数据库
3. 运行 `main('dashboard')` 生成QC报告

### 算法开发者：添加新分析模块
1. 在 `Functions/Statistics/` 中创建新函数
2. 所有新参数写进 `config.m`
3. 更新 `main.m` 和 `runPhase2.m`
4. 运行 `main('phase2')` 验证输出

### 代码审查者：运行科学审核
```matlab
run('Review/runPhase2Review.m')
```
会生成完整的10级审核报告。
